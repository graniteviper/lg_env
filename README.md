# Liquid Galaxy Lab GSOC 2026 - Task 2

## Description
- This is a repository for the task-2 of liquid galaxy lab for GSOC 2026.
- This project is best run on an android emulator and make sure that you have the latest version of flutter installed in your machine.

## Setting it up
1. Clone the repo from github.
2. CD into the folder created using the `cd` command. 
3. Run `flutter pub get` in the folder.
4. Run `flutter doctor` in the folder.
5. Make sure you have android emulator running to try out the application.
6. Run `flutter run` to see the application in action.

## How to use ?
1. Wait for the app to start up, you should see a splash screen and then it will lead you to the home screen.
2. Click on the connection button and enter the credentials of your machine.
3. Click back and now you can use the application.
